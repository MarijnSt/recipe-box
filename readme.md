A project I'll use to learn about Django, Django Rest Framework, Vue and TypeScript.

## Project setup
### Backend
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Frontend
```bash
cd frontend
nvm use
npm install
```

## Run the project
### Launch backend
```bash
python manage.py runserver
```
### Launch frontend
```bash
cd frontend
nvm use
npm run dev
```
